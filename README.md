# 编译方法

直接运行得到结果
```shell
$ make r
```

**重要**：需要flex工具，还有bison工具

下面的命令生成词法分析器
```shell
$ make lb
```

使用下面的命令，运行单元测试，并给出两个演示文件文法分析结果
```shell
$ make ut
```


使用如下命令来清理编译结果
```shell
$ make clean || make clear
```
